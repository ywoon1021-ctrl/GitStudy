git 명령어

0. 터미널 => new 터미널 1.git init(git 열기?) 2.git add + "파일명" (파일 깃에 추가) (파일고르기) (staging area_commit할 파일을 골라놨다는 것) 2_1. git add app.html app2.html => 여러 파일 저장 2_2. git add . 모든파일 스테이징 3.git commit (기록 명령) , -m (메모, 혹은 버전) (파일을 깃 폴더에 넣기) (repository_저장소) 4.git status (상태창) (현재 스테이징되어있는 파일조회) 5.git log --all --oneline (현재 커밋 되어있는 파일 조회)
